# BoxDefense Core Features

1. **Multi-Role RBAC**: Strict access control for Customers, Moving Companies, and Admins.
2. **Photo Documentation**: Integration with cloud storage and automatic thumbnailing for legal evidence.
3. **QR System**: Unique encrypted QR codes generated for every box for integrity validation.
4. **Legally Binding Records**: Generate text-based liability documents and CSV inventory exports.
5. **Inventory Precision**: Track weight (lbs), count types, and detailed item descriptions.
6. **Admin Oversight**: Monitor global system health, manage roles, and flag suspicious entities.
7. **Read-Optimized Cache**: Server-side JSON caching for sub-millisecond inventory retrieval.
